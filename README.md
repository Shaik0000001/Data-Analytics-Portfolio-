# Data Analytics Internship Portfolio

## 📌 Overview
This project demonstrates an end-to-end data analytics workflow using an E-commerce dataset.

## 🎯 Objectives
- Clean and prepare raw data
- Perform exploratory data analysis (EDA)
- Generate business insights
- Build dashboards and KPIs
- Apply statistical validation

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL
- Power BI

## 📊 Key Insights
- Electronics category generates the highest revenue
- UPI payments have higher average order value than COD
- Majority of customers are one-time buyers
